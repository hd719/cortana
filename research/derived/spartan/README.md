# Spartan Derived Research

This directory is the first compiled layer on top of the raw Spartan paper corpus.

Use it for:
- bucket summaries
- evidence maps
- question-driven synthesis
- candidate rules that may later be promoted into Spartan knowledge or planning docs

Do not treat these files as current truth by default. They are research syntheses, not canonical policy.
