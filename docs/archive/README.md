# Cortana Archive

This folder holds docs that are still worth keeping in git but should not be treated as the active docs set.

Use this for:

- historical research
- deep implementation notes
- superseded or low-signal design docs
- older planning notes that are no longer front-door material

Rules:

- do not create new active docs here
- do not link to archive docs as canonical current truth unless there is no active replacement
- move docs here when they are still useful reference but make the main docs tree too noisy
