# Spartan Roadmap

The Spartan roadmap is effectively complete at the planning level and should now be read as a summary of the implemented program shape.

## Program Goal

Build a trustworthy coaching loop across Tonal, Whoop, nutrition, and coaching outputs.

The strongest current principles are:

- data trust must come before higher-order coaching
- volume and recovery logic should be evidence-backed
- confidence should degrade when coverage is weak
- Apple Health is a later expansion, not the starting point

## Implemented Epic Order

1. Core baseline
2. Training intelligence
3. Tonal programming
4. Coaching operating loop
5. Health expansion

Those detailed planning triplets are archived because they are already implemented.

## What The Roadmap Says About Current Priorities

The current live emphasis should stay on:

- provider and data correctness
- canonical athlete-state quality
- trustworthy nutrition coverage
- credible workload and progression logic
- usable coaching outputs rather than speculative new surfaces
