# Tonal PPL v1 Research Audit

- Reviewed: 2026-04-08
- Scope: validate `tonal-ppl-v1` against current hypertrophy evidence, your real Tonal history, and high-signal bodybuilding coaching guidance

## Bottom Line

This is a strong `v1` for you, not a magical perfect plan.

Why it holds up:
- the exercise list is Tonal-valid instead of invented
- the base movements are drawn from your own repeat history, not generic bodybuilding templates
- the split covers the main hypertrophy patterns cleanly:
  - push: horizontal press, vertical press, chest isolation, delt isolation, triceps
  - pull: row, pulldown, lat accessory, biceps, trunk
  - legs: single-leg quad pattern, hinge, frontal-plane glute work, trunk

Main caveat:
- the literature supports the current movement selection and rep structure well
- the part that still needs real-world tuning is weekly dose and scheduling against your running and recovery

## What The Evidence Supports

### 1. The split itself is acceptable

The current literature supports that total weekly volume matters more than whether the split is called PPL, upper/lower, or full body when volume is matched.

Practical meaning for you:
- PPL is fine
- what matters is that the weekly hard sets are recoverable and repeated consistently
- if possible, a rolling PPL cadence is better than only hitting each pattern once every 7 days

### 2. The main lift choices are solid

The plan uses efficient multi-joint anchors:
- `Bench Press`
- `Standing Barbell Overhead Press`
- `Barbell Bent Over Row`
- `Neutral Grip Lat Pulldown`
- `Barbell Front Rack Split Squat`
- `Barbell RDL`

That lines up well with time-efficient hypertrophy guidance: build the session around a leg pattern, an upper push, and an upper pull, then add targeted accessory work.

### 3. The accessory work is justified

The current bodybuilding and hypertrophy guidance does support isolation work when the goal is muscle gain, especially for:
- chest variation
- arms
- shoulders
- trunk stability / torso control

For you specifically, direct arms should stay in because your observed Tonal history shows repeated successful use of:
- `Triceps Extension`
- `Overhead Triceps Extension`
- `Barbell Biceps Curl`

Those are not random add-ons. They are movements you already use on the machine.

### 4. The rep targets are appropriate

The current `6-10` compound and `10-15` accessory structure is defensible.

That fits the hypertrophy literature well:
- compounds can grow muscle effectively in lower-to-moderate rep ranges
- accessories work well in moderate-to-higher rep ranges
- the key is getting sufficiently hard sets while managing fatigue

### 5. Failure is not required

This plan should not be run as a constant failure plan.

Best operating rule:
- compounds: mostly `1-2 RIR`
- accessories: `0-2 RIR`

That fits the current evidence better than making every hard set an all-out grinder.

### 6. Concurrent running matters

Because you run and lift, the interference question is real.

The current plan is still workable, but the lower day should respect run load:
- avoid stacking hard runs right next to the leg day when possible
- if run strain is already high, keep the lower day to the anchors and trim one accessory slot

## Plan-Specific Recommendations

### Keep As-Is

- `Bench Press` as the main push anchor
- `Standing Barbell Overhead Press` as the main vertical press
- `Barbell Bent Over Row` and `Neutral Grip Lat Pulldown` as the pull anchors
- `Barbell Front Rack Split Squat` and `Barbell RDL` as the leg anchors
- direct biceps and triceps work

### Good Alternates

Use these as swaps when joints, boredom, or fatigue call for a small change:
- `Incline Chest Fly` or `Middle Chest Fly` instead of `Decline Chest Fly`
- `Barbell Seated Overhead Press` instead of `Standing Barbell Overhead Press`
- `Standing Chop` instead of `ISO Split Squat Lift`

### What Still Needs Live Tuning

- whether `Decline Chest Fly` or `Incline Chest Fly` feels better on shoulders and gives better chest stimulus for you
- whether both triceps slots should stay in every push day or whether the second one should remain optional most weeks
- whether pull day needs one more rear-delt / upper-back option later after more Tonal mapping coverage exists
- whether leg day is best kept at 4 movements or should occasionally rotate in a lighter lower-fatigue version after hard run blocks

## Recommended Operating Rules

- Run the plan as a rolling split when possible instead of a rigid once-per-week body-part schedule.
- Keep compound rest periods around `2-3 minutes`.
- Keep accessory rest periods around `60-120 seconds`.
- If Whoop recovery is yellow or red, trim one accessory before trimming the main lift.
- If the day already includes a hard run, do not also force a maximal lower-body session.

## Verdict

For your current Tonal setup, this is a high-confidence PPL `v1`.

It is good enough to run now because it is:
- Tonal-valid
- grounded in your own history
- consistent with current hypertrophy evidence

What makes it not yet "perfect" is not exercise selection. It is calibration:
- weekly set dose
- exercise tolerance
- fatigue interaction with running
- which fly / arm / core variations feel and perform best for you
